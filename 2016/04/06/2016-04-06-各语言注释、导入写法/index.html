发表于 2016年04月06日 | 更新于 2018年06月23日 | 分类于 技术比较

常见语言注释写法

行注释

// 注释内容

C、C++、C#、Go、Java、JavaScript、LESS、Objective-C、PHP、SASS、Swift

# 注释内容

Perl、PHP、Python、Ruby、Shell

-- 注释内容

Haskell、Lua、SQL

; 注释内容

Assembly、Lisp、Scheme

% 注释内容

LaTeX、MATLAB

‘ 注释内容

Visual Basic

块注释

/* 注释内容 */

C、C++、C#、CSS、Go、Java、JavaScript、LESS、Objective-C、PHP、SASS、Swift

#| 注释内容 |# (= =不加点东西这个#显示不出来)

Lisp、Scheme

=begin 注释内容 =cut(=end in Ruby)

Perl、Ruby

{- 注释内容 -}

Haskell

<!-- 注释内容 -->

HTML

--[[ 注释内容 ]]

Lua

%{ 注释内容 %}

MATLAB

‘’’ 注释内容 ‘’’

Python

“”” 注释内容 “””

Python

<# 注释内容 #>

Shell
