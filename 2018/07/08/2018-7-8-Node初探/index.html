创建一个 Node 服务器

新建一个空项目文件夹，然后创建一个 demo.js 文件

在Visual Studio Code中侧边栏选择调试，配置选择 Node.js 环境

这时候会弹出配置文件，或者手动对该目录下.vscode文件夹下的launch.json文件进行配置

先Shift+~调出调试控制台，然后点击调试按钮，就可以成功看到成功启动服务器啦

然后在浏览器中访问该路径，即发送一个 GET 请求，就可以看到返回结果啦
